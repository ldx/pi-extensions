# pi-tools-manager

Pi extension that adds `/tools` for inspecting and toggling active tools.

## Commands

- `/tools` -- show active and available tools.
- `/tools <tool-name>` -- show details for a tool.
- `/tools enable <tool-name...>` -- enable tools.
- `/tools disable <tool-name...>` -- disable tools.
- `/tools toggle <tool-name...>` -- toggle tools.
- `/tools reset` -- enable all tools.
